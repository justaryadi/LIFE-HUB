# LIFE-HUB

Saat ini banyak orang menggunakan berbagai aplikasi berbeda untuk mengatur kehidupannya, seperti aplikasi to-do list, kalender, notes, focus timer. Penggunaan terlalu banyak aplikasi justru membuat kehidupan digital menjadi semakin kompleks dan tidak terorganisir.
Selain itu, gaya hidup modern yang serba cepat juga menyebabkan masyarakat semakin kurang memperhatikan kebiasaan kecil yang berdampak terhadap lingkungan, seperti penggunaan plastik berlebihan, pemborosan energi, dan pola konsumsi yang kurang sustainable.

Berdasarkan permasalahan tersebut, dibuatlah sebuah website bernama LIFEHUB, yaitu platform digital berbasis web client yang berfungsi sebagai pusat pengelolaan kehidupan sehari-hari secara modern, terstruktur, interaktif, dan mindful.
LIFEHUB tidak hanya membantu pengguna meningkatkan produktivitas dan keseimbangan hidup, tetapi juga mengajak pengguna membangun kebiasaan positif terhadap lingkungan melalui konsep sustainable side quest dan sistem achievement berbasis gamification.
